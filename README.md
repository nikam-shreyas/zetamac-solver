## Zetamac Solver

This project aims to automate the process of solving quantitative questions found on the Zetamac website. It utilizes Selenium for interacting with web browsers and automating navigation.


### Setup

1. Install the required libraries:

```bash
pip install selenium
```

2. Download the appropriate ChromeDriver for your Chrome version from: [https://chromedriver.chromium.org/downloads](https://chromedriver.chromium.org/downloads)

3. Place the ChromeDriver executable in your system PATH or alongside the project script.

### Usage

1. Run the main script (`solve_questions.py`).
```
python3 solve_questions.py
```
2. The script will automatically solve the questions and display the results.


### Disclaimer

This project is for educational purposes only. It is crucial to ensure the accuracy of the extracted expressions and interpret the results with caution. The project's effectiveness may vary depending on the website's structure and complexity.
